# Used Products Buying and Selling Website

## About Projects

This website is for buying and selling used laptops. Here a seller can add his product and advertise any product if needed. If there is a problem with any product , the buyer can report it, which the admin can take action on.

### Prijects Features:

- You can buy your favorite products here.
- A seller can add his/her products for sale.
- Admin can control the website


## Uses technology:

- Tailwindcss
